LSB Steganography

What is Steganography?

Steganography is the practice of concealing a message within another file or message to avoid detection. Unlike cryptography, which aims to make a message unreadable, steganography aims to hide the existence of the message itself.

How LSB (Least Significant Bit) Steganography Works

LSB steganography works by replacing the least significant bit of each byte in an image file with bits from the secret message. Since the least significant bit has the smallest impact on the color value, changes are virtually imperceptible to the human eye.

The Process

For example, if we have the letter 'A' (ASCII 65, binary 01000001) to hide in an image:

Original pixel: [00101101, 00011100, 11011100]
After hiding: [00101100, 00011101, 11011100]

Only 2 bits needed to change, making the alteration virtually undetectable.

Features
  • Virtually imperceptible changes to the image
  • Works with any type of message (text, binary data)
  • No change in file size, avoiding suspicion
  • Simple yet effective technique
Limitations
  • Limited capacity (typically 1/8 of the file size)
  • Not resistant to image manipulation or compression
  • Works only with lossless formats (PNG, BMP, TIFF)
  • Vulnerable to statistical analysis
Important Notes
  • Always use lossless image formats (PNG, BMP, TIFF)
  • JPEG/JPG compression will destroy hidden data
  • The image must be large enough to hold your message
  • Keep your encoded images secure
